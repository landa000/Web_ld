## git 사용법

1. github 가입
2. new repository 생성
3.  git 자료 복제
   -  내 컴퓨터에 git clone 생성한 repository
4. README.md 파일 생성 및 작성
5. github으로 push
   - 자료를 올린 사람의 이름+이메일
   - 계정 id/pw
6. github로 push처리하여 보내기
7. github의 자료를 pull하여 가져오기
