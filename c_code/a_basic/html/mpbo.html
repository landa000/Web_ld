margin, padding 쓰임새

하나의 요소안에 첫번째 요소가 float,position 처리가 없을시, margin-top값을 사용할 경우엔 부모요소를 margin 처리하게된다

margin값이 위아래 겹칠 경우, 큰값만 처리됨(단, float처리된 겹침은 제외)

margin값은 눈에 보이는 공간의 거리값 조절기능, +가 아닌 -값도 처리된다.(padding은 안됨)

단, margin값은 auto, 좌우에 한해서만 %처리 가능, 상하에 %값 처리는 지양해야함
상하는 명확한 값을 파악하기 어려움(부모의 높이가 기준이 아님)

float기능과 margin 기능을 같이 사용할 경우,margin:auto;는 좌/우 가운데 배치되지 않는다
